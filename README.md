# Unique Attendance Tracker (Python)

A simple Python program that records student attendance and ensures only unique roll numbers are counted using a set.

## Features
- Takes roll numbers as input
- Stores attendance without duplicates
- Displays total number of present students
- Shows the list of present students

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository
2. Run the program:
   python attendance_tracker.py
